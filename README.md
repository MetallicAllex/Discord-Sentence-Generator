# 使用方法
在 **发送消息** 之前，点开右下角的设置，在里面：<br />设定 **【用户名、头像和是否显示某公司图标】** 后即可发送消息。

# 修改方法
换掉图标，并且修改 `script.js` 内核图标有关的东西即可。
